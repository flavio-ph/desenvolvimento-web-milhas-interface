# MilhasPro - Gestão Inteligente de Milhas

Bem-vindo ao **MilhasPro**, uma aplicação web moderna para gestão de pontos e milhas aéreas. Este projeto foi desenvolvido para ajudar usuários a controlar seus programas de fidelidade, acompanhar saldo de milhas, registrar compras bonificadas e aproveitar as melhores promoções.

## Tecnologias Utilizadas

Este projeto foi construído com as seguintes tecnologias principais:

-   **[React](https://react.dev/)**: Biblioteca JavaScript para construção de interfaces de usuário.
-   **[TypeScript](https://www.typescriptlang.org/)**: Superset do JavaScript que adiciona tipagem estática.
-   **[Vite](https://vitejs.dev/)**: Build tool rápida e leve para desenvolvimento frontend.
-   **[Tailwind CSS](https://tailwindcss.com/)**: Framework CSS utility-first para estilização rápida e responsiva.
-   **[React Router DOM](https://reactrouter.com/)**: Biblioteca para gerenciamento de rotas na aplicação.
-   **[Lucide React](https://lucide.dev/)**: Biblioteca de ícones leve e consistente.
-   **[Recharts](https://recharts.org/)**: Biblioteca para criação de gráficos e visualização de dados.
-   **[Axios](https://axios-http.com/)**: Cliente HTTP para requisições à API.

##  Funcionalidades

### Área Pública
-   **Login e Registro**: Autenticação de usuários.
-   **Recuperação de Senha**: Fluxo para redefinição de credenciais.

### Área do Usuário (Dashboard)
-   **Visão Geral**: Resumo de pontos, milhas a expirar e valor estimado.
-   **Gestão de Cartões**: Cadastro e visualização de cartões de crédito.
-   **Registro de Compras**: Lançamento de compras para controle de pontuação.
-   **Histórico**: Extrato detalhado de transações e movimentações.
-   **Promoções**: Visualização de promoções vigentes de transferência e acúmulo.
-   **Perfil**: Gerenciamento de dados pessoais.
-   **Notificações**: Central de avisos para o usuário.
-   **Busca**: Pesquisa global no sistema.

### Área Administrativa
-   **Gestão de Marcas**: Administração das marcas parceiras.
-   **Gestão de Programas**: Controle dos programas de fidelidade disponíveis.
-   **Cadastro de Promoções**: Criação e edição de promoções para os usuários.

## Pré-requisitos

Antes de começar, certifique-se de ter instalado em sua máquina:
-   [Node.js](https://nodejs.org/) (versão recomendada: 18 ou superior)
-   npm (gerenciador de pacotes do Node)

## Instalação

1.  Clone o repositório (se aplicável) ou navegue até a pasta do projeto:
    ```bash
    cd desenvolvimento-web-milhas-interface
    ```

2.  Instale as dependências do projeto:
    ```bash
    npm install
    ```

## Executando o Projeto

Para iniciar o servidor de desenvolvimento:

```bash
npm run dev
```

A aplicação estará disponível em `http://localhost:5173` (ou outra porta indicada no terminal).

## Scripts Disponíveis

-   `npm run dev`: Inicia o servidor de desenvolvimento.
-   `npm run build`: Compila a aplicação para produção.
-   `npm run preview`: Visualiza a versão de produção localmente.

## 🎨 Tema

O projeto conta com suporte a **Tema Escuro (Dark Mode)** e **Tema Claro (Light Mode)**, adaptando-se à preferência do sistema ou seleção do usuário, utilizando as classes do Tailwind CSS.
